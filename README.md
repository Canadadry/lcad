# lcad

A minimal 3D modeling and texturing tool built with Lua / LÖVE (Love2D).

Load, view, and edit low-poly `.obj` meshes (vertices, faces, UVs), then paint
their texture with a fixed palette — all in one small app.

## Requirements

- [LÖVE](https://love2d.org/) (Love2D)

## Usage

```
make run    # launch the app
make test   # run the test suite
make build  # package into lcad.love
```
